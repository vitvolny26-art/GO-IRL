# 🚀 GO IRL Development Setup - Автоматическая Установка

## Быстрый старт (3 шага)

### Шаг 1: Запустить установку (как администратор)

```powershell
# Откройте PowerShell от администратора и выполните:
cd C:\Users\lenovo\Documents\Codex\2026-05-28\new-chat
.\setup-dev-tools.ps1
```

Или просто:
1. Найдите файл `setup-dev-tools.bat` в папке проекта
2. Кликните правой кнопкой → "Запустить от имени администратора"

### Шаг 2: Установить Chrome расширения

Скрипт выдаст 3 ссылки. Откройте каждую в браузере и нажмите "Добавить расширение":

1. **Shotdiff** — для визуального сравнения до/после
2. **React DevTools** — для отладки React компонентов
3. **Live.js** — для автоперезагрузки

### Шаг 3: Запустить dev сервер

Найдите `start-dev.bat` в папке проекта и двойной клик.

Или в PowerShell:
```powershell
cd C:\Users\lenovo\Documents\Codex\2026-05-28\new-chat
pnpm run dev
```

## Что устанавливается

- ✅ **ESLint** — проверка ошибок

### Node Пакеты
- ✅ Все зависимости из `package.json`

### Startup Скрипты
- ✅ `start-dev.bat` — быстрый запуск dev сервера
- ✅ `.env.local` — конфиг окружения

## Как это работает

```
1. Ты открываешь VS Code
   ↓
2. Редактируешь код (например, добавляешь новый компонент)
   ↓
3. Нажимаешь Ctrl+S (сохранить)
   ↓
4. Vite автоматически пересобирает проект
   ↓
5. Браузер автоматически обновляется (hot reload)
   ↓
6. Shotdiff показывает все визуальные изменения
   ↓
7. React DevTools показывает изменения компонентов
```

## Если что-то не сработало

### VS Code не установлен
```powershell
winget install Microsoft.VisualStudioCode
```

### pnpm не установлен
```powershell
npm install -g pnpm
```

### Нужны права администратора
1. Откройте PowerShell
2. Правой кнопкой → "Запустить от имени администратора"
3. Выполните скрипт заново

### Порт 5178 уже занят
```powershell
# Найдите процесс на этом порту
netstat -ano | findstr :5178

# Или просто используйте другой порт
pnpm run dev -- --port 5179
```
