# Roadmap: GO IRL

## Sprint 1 (MLP)

Minimum Lovable Product.

- 🔐 Telegram Login
- 👤 User Profile
- 📍 Browse Activities
- ➕ Create Activity
- ✅ Join Activity
- 💬 Activity Chat
- 📅 Add to Calendar

**Goal:** First users can discover, create, and join real activities.

## Sprint 2

- 📍 Geo-verification of participation
- ⭐ Reviews and ratings (v1)
- 🤝 Trust Score (simple version)

**Goal:** Build trust through verified participation.

## Sprint 3

- 🏐 Sport module excellence
- Refined UX based on real user feedback

**Goal:** One vertical (Sport) becomes best-in-class.

## Future

- Additional verticals (Nature, Learning, Creative)
- Marketplace features
- Organization support
- AI recommendations
- Mobile apps

## Success Criteria

Not DAU. Not feature count.

**Only metric:** 100 confirmed real-life meetings.
