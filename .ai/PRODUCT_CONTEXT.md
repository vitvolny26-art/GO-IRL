# GO IRL Product Context

## North Star

**Confirmed Real-Life Meetings**

Not DAU. Not session time. Only meetings that actually happened.

## Mission

Help people spend less time online and more time together.

**Go outside. Meet people. Live more.**

## Core Product Loop

1. User discovers an activity
2. User joins the activity
3. User meets people in real life
4. Friendship/community forms
5. User organizes more activities

If this loop breaks at any step, the product fails.

## MLP Features (Sprint 1)

- 🔐 Telegram Login
- 👤 User Profile
- 📍 Browse Activities
- ➕ Create Activity
- ✅ Join Activity
- 💬 Activity Chat
- 📅 Add to Calendar

**Nothing else. No exceptions.**

## Success Criteria

After Sprint 1:
- 50 real users in Olomouc
- First 20 users invited personally
- At least 10 confirmed real-life meetings
- Users would be sad if GO IRL disappeared

## Geographic Focus

Phase 1: Olomouc only
Phase 2: Czech Republic
Phase 3: Europe

## Technology Stack

- **Backend:** Node.js + Fastify + TypeScript
- **Database:** PostgreSQL (Supabase)
- **ORM:** Prisma
- **Frontend:** React (web), Telegram Mini App
- **Realtime:** WebSocket
- **Auth:** Telegram OAuth2
